Rule 01
📦 The Box Grid
A 4×5 grid of 20 boxes. One hides the cat — the rest hold GATE questions. You cannot see which box the cat is in (quantum superposition).

✓ Cat Box → Instant Level Win!
✗ Wrong Box → Question appears (no oxygen loss from opening)
Rule 02
🌬️ Oxygen System
Starts at 100% O₂. Opening a wrong box does NOT reduce oxygen.
Wrong answer: −5%
Correct answer: +10%
Upload proof: +25%

O₂ is capped at 100%. Reach 0% → game over!
Rule 03
🐾 Cat Movement
Every 60 seconds, the cat moves to an adjacent box (horizontal/vertical). It will never move into an already-opened box.

⚠ Plan quickly — the cat is always moving!

When you upload proof for a question, the cat returns to its previous position.
Rule 04
🧠 GATE Questions
65 subject-based questions per session (GATE 2021–2025, including General Aptitude section). Questions are randomised each session. You must answer every question — unanswered questions will loop back until correct.

You must answer before opening the next box.
Rule 05
✅ Multi-Answer Questions
Some questions accept multiple correct answers. If you select even one correct option, the answer is treated as correct.

After answering, all correct answers are highlighted so you can learn the full solution.
Rule 06
📤 Upload Proof
Use the Scratchpad to work out your answer — draw, type notes, or upload an image/PDF as proof.

Uploading a file adds +25% O₂ and returns the cat to its previous box.

Tap "Open Google Keep" to use Keep for longer notes.
Rule 07
💾 Auto-Save
The game saves your subject, cat, oxygen, level, and questions answered to local storage after every action.

Close and return? You'll be offered to Resume exactly where you left off.

Note: Clearing browser data erases saved progress.
